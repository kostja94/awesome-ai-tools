# Dalle 3 by OpenAI

## Product Information

**Name**: DALL-E 3

**Website**: https://openai.com/index/dall-e-3/

**Category**: AI Image Tools

**Description**: 
OpenAI 开发的第三代 DALL-E 模型，能够根据文本描述生成高质量、创意丰富的图像。

## Key Features

- 文本到图像生成
- 高质量图像输出
- 图像编辑功能
- 与 ChatGPT 集成
